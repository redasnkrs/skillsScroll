# Itinéraire des trophées

### Configuration générale

- Personnage : Tav entièrement personnalisé (pas d’Origin) en mode Équilibré pour la première partie. Sauvegardes manuelles fréquentes.
    
- Trophées de compagnons & romances : commencez tôt ; évitez de vous reposer lorsque des scènes sensibles sont en attente. Gardez une sauvegarde séparée avant les décisions majeures.
    
- Point de non-retour : attention à l’avertissement du Gouffre de Shar à l’Acte 2 ; terminez tout le contenu annexe avant d’aller plus loin.
    

## Acte 1 : Nautiloïde → Bosquet d’Émeraude → Outreterre/Col du Mont

1. Prologue : Nautiloïde
    
- [ ] Devil’s in the Details : tuer le commandant Zhalk sur la passerelle avant d’interagir avec les commandes ; utilisez Commandement de Shadowheart pour lui faire lâcher la Lame Éternelle.
        
- [ ] Descent From Avernus : terminer le prologue en pilotant le vaisseau.
        
- [ ] Optionnel : secourir Shadowheart ; recruter “Us” pour de l’aide.
        
2. Camp/Systèmes dès le début
    
- [ ] Bedrolls and Breakfast : faire quatre longs repos au cours de la partie.
        
- [ ] Expand Your Mind : consommer un “Spécimen de parasite flagelleur mental”.
        
- [ ] Outsourcing : libérer Withers (Crypte humide), puis recruter un mercenaire.
        
- [ ] Roleplayer : progresser naturellement les objectifs de l’arrière-plan en explorant et en accomplissant des quêtes.
        
3. Bosquet d’Émeraude & Scratch/Cub d’ours-hibou
    
- [ ] You Have Two Hands for a Reason : caresser Scratch et le cub d’ours-hibou ensemble après leur arrivée au camp (ne pas utiliser de potion Parler aux animaux pour les caresser).
        
- [ ] Fetch Quest : jouer à la balle avec Scratch (événement de balle au camp).
        
- [ ] She Cannot be Caged! : commencer la chaîne de Sazza : la libérer de la prison du bosquet sans combat ; l’escorter par la route secrète ; plus tard, empêcher sa mort au camp gobelin. Ceci se conclut à l’Acte 2.
        
4. Village flétri & Profondeurs
    
- [ ] Pest Control : tuer la Matriarche araignée avant l’éclosion des œufs dans les Profondeurs murmurantes.
        
- [ ] Piller l’Améthyste sombre et autres secrets ; garder des pelles pour les coffres enterrés (Dig for Victory).
        
5. Grymforge & Forge
    
- [ ] Forged in Blood and Fire : utiliser la Forge adamantine pour fabriquer un objet.
        
- [ ] A Grym Fate : tuer Grym sans utiliser le marteau de la forge (l’attirer dans la lave, couper le marteau).
        
- [ ] Préparer La colère de la Reine Liche en s’alliant plus tard à Voss selon les voies choisies.
        
6. Divers classe/combat
    
- [ ] Action Surge : guerrier avec Attaques supplémentaires + Rush d’action pour atteindre cinq attaques en un tour (Hâte aide).
        
- [ ] Fists of Fury : tuer un ennemi avec une attaque à mains nues.
        
- [ ] Kill Two Birds with One Gnome : utiliser l’action “Arme improvisée” verte pour lancer un ennemi sur un autre.
        
7. Col du Mont / Crèche / Rosymorn
    
- [ ] Taking Blood : voler le Sang de Lathandre sous le Monastère de Rosymorn.
        
- [ ] Rude, Crude, and full of Attitude : trouver et invoquer le quasit (sous-sol de l’apothicaire, Village flétri).
        
- [ ] Débuter la romance de Karlach pour Hot Date plus tard.
        
8. Quitter l’Acte 1
    
- [ ] The Plot Thickens : passer à l’Acte 2 via Outreterre ou Col du Mont.
        

## Acte 2 : Terres en Ombre → Moonrise → Gouffre de Shar

1. Avertissements critiques
    
- [ ] Nettoyez l’Auberge de la Dernière Lumière, Moonrise et les quêtes périphériques tôt. L’avertissement du Gouffre de Shar marque un large verrouillage au-delà.
        
2. Progression cœur de l’Acte 2
    
- [ ] Repairing the Weave : stabiliser l’orbe de Gale.
        
- [ ] Under Lock and Key : libérer tous les prisonniers de Moonrise en une partie.
        
- [ ] Non-invasive Procedure : tuer le Chirurgien avant qu’il n’opère sur vous.
        
- [ ] Penny Pincher : vaincre la Perceptrice de péage sans qu’elle utilise l’or.
        
- [ ] No Free Lunches : battre l’Apôtre de Myrkul avant qu’il ne consomme des Nécromites.
        
- [ ] To Bloom in Darkest Night : offrir une orchidée nocturne à Shadowheart.
        
3. Chaînes en cours
    
- [ ] She Cannot be Caged! : finaliser le sauvetage de Sazza à Moonrise.
        
- [ ] Leave No One Behind : assurer la survie de tous les réfugiés tieffelins sur les Actes (requiert des choix prudents depuis l’Acte 1).
        
4. Sortie de l’Acte 2
    
- [ ] The City Awaits : terminer la ligne de Ketheric et partir vers la Porte de Baldur.
        

## Acte 3 : Ville basse → Wyrm’s Rock → Finales

1. Mise en place & romances
    
- [ ] Busker : gagner 100 pièces d’or via des performances de barde (rapidement à travers les foules de Rivington à la Ville basse).
        
- [ ] Hot Date : compléter la romance de Karlach à l’Acte 3.
        
- [ ] Mind Blown : romancer l’Empereur via des choix illithids/contexte.
        
2. Trophées de boss/rencontres
    
- [ ] Fancy Footwork : vaincre Gortash à Wyrm’s Rock sans déclencher de pièges.
        
- [ ] First Blood : tuer Orin pendant que les cultistes chantent.
        
- [ ] Interfectorem Draconis : tuer le dragon rouge dans la Ville haute.
        
- [ ] Crash Landing : dans Wyrmway, abattre le dragon en plein vol.
        
3. Grandes voies & finales
    
- [ ] Murder in Baldur’s Gate : devenir Assassin impie de Bhaal via la chaîne de Sarevok.
        
- Finales :
        
- [ ] Hero of the Forgotten Realms : détruire l’Absolu.
            
- [ ] Absolute Power Corrupts : contrôler le Cerveau-Néant.
            
- [ ] Ceremorphosis : devenir flagelleur mental pour vaincre le Cerveau-Néant.
            
4. Nettoyage de trophées (tout acte si manqué)
    
- [ ] Homebrewer : fabriquer trois solutions alchimiques uniques.
        
- [ ] Escapologist : se faire arrêter ; s’évader de prison.
        
- [ ] No Penny Required : réussir un test de Détection des pensées.
        
- [ ] Bookworm : lire 100 livres différents.
        
- [ ] Bottoms Up : Long repos en n’utilisant que de l’alcool.
        
- [ ] Shove Off : tuer une créature par chute.
        
- [ ] Jack-of-all-Trades : multiclasser dans chaque classe sur un même personnage jusqu’au niveau 12 ; éviter d’utiliser le respec de Withers avant le déblocage.
        

## Deuxième partie (courte) : **Dark Urge** en Tacticien

- Difficulté : Tacticien dès un nouveau départ.
    
- Trophées ciblés :
    
- Embrace Your Urge : devenir le Tueur de Bhaal.
        
- Sins of the Father : revendiquer le Cerveau-Néant pour Bhaal.
        
- Critical Hit : terminer le jeu en Tacticien.
        

## Notes sur les manquables/verrouillages (mémo)

- Avertissement du Gouffre de Shar : considérez-le comme un point de non-retour ; terminez le contenu annexe avant d’avancer.
    
- Chaîne de Sazza : doit être gérée sans violence ; se conclut sur les Actes 1–2.
    
- Réfugiés tieffelins : des choix du Bosquet à l’Acte 2 affectent Leave No One Behind.
    
- Zhalk : doit être tué au prologue avant l’interaction avec la passerelle pour le trophée/l’arme.
    
- Grym (sans marteau), Matriarche (sans œufs) : conditions de mise à mort spécifiques.
    

## Structure Obsidian suggérée

- Temps estimé pour le platine : 150–200 heures sur deux parties.
    
- Impact principal de difficulté : Critical Hit uniquement (Tacticien).
